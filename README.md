# BaldPhone
Platform which replaces the phone's interface to a big, simple and friendly one.

Main features:
 - Home Launcher
 - Keyboard
 - Video Tutorials
 - Reminders For Pills
 - Contacts and Phone Applications
 - Voice Assistant
 - Alarms
 - Photos and Videos Viewer
 - Notification Management
 - SOS Button
 - Advanced Settings
